# 客户跟进提醒系统

## 快速开始

### 方式一：双击运行桌面应用（推荐）

```
双击 → start.bat → 选择 [1] 桌面应用模式
```

系统会打开一个独立的桌面窗口，就像普通 Windows 软件一样。

### 方式二：浏览器访问

```
双击 → start.bat → 选择 [2] 浏览器模式
```

浏览器会自动打开 http://localhost:8080

---

## 第一次使用

1. 打开系统后，点击顶部「📊 上传 Excel」按钮
2. 选择你的客户跟进表格（.xlsx 格式）
3. 系统自动导入所有客户数据并生成跟进提醒

**表格需要包含的列**：客户名称、国家、客户等级、客户类型、状态等

## 功能一览

| Tab | 说明 |
|-----|------|
| 仪表盘 | 客户统计、待办提醒、快捷操作 |

| 新客户池 | 新导入待联系的客户 |
| 回收站 | 已删除的客户，可恢复或永久删除 |
| 设置 | Excel 同步、上传表格、系统配置 |
| 操作日志 | 查看所有操作记录 |

## 分享给同事

### 打包桌面版（推荐，无需 Python）

```
步骤 1：双击运行「构建EXE.bat」→ 等待 3-5 分钟
步骤 2：双击运行「打包分享.bat」→ 生成 ZIP 文件
步骤 3：把 ZIP 发给同事
```

同事收到后只需：解压 → 双击 exe → 上传自己的表格 → 完成。

**优点**：无需安装 Python，双击即用。

### 打包源码版（需 Python 3.9+）

```
双击「打包分享.bat」→ 选择 [S] 源码版 → 生成 ZIP
```

## 常见问题

**Q：数据库在哪里？**
A：打包版自动保存在 `%APPDATA%\客户跟进提醒系统\data\`，源码版在 `data/` 目录。

**Q：每个同事的数据会互相影响吗？**
A：不会。每个人的数据存在自己电脑上，完全独立。

**Q：需要联网吗？**
A：不需要。所有数据都在本地存储。
